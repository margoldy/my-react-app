# Система голосования
Простое React-приложение для проведения голосования с визуализацией результатов.
###[Посмотреть работающее приложение](https://yourusername.github.io/your-repo-name)

## Функциональность
### 1. Возможность голосования (выбора варианта)
Приложение позволяет выбирать один из трёх предложенных вариантов. Выбранный вариант подсвечивается и масштабируется, появляется кнопка "проголосовать."

### 2. Вывод результатов
После нажатия на кнопку открывается вторая страница и отображается количество голосов для каждого варианта. Результаты визуализируются с помощью прогресс-баров и суммарного числа проголосовавших.

### 3. Сохранение данных
Все данные автоматически сохраняются в браузере (Local Storage), при перезагрузке страницы восстанавливают результаты голосования и текущее состояние.

### 4. Управление
- Проголосовать (на стартовой странице): Учитывает голос и открывает страницу результатов
- Возврат к голосованию: Кнопка для возврата на выбор варианта
- Сброс результатов: Возможность обнулить все голоса и начать заново

## Использование
1. Выберите понравившийся вариант из трех предложенных
2. Нажмите кнопку "Проголосовать"
3. Просмотрите результаты голосования
4. При необходимости вернитесь к голосованию или сбросьте результаты
